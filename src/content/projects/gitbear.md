A native Android app for browsing GitLab. Supports OAuth login, repository and merge request browsing, and in-app Markdown rendering via Markwon. Built with Kotlin using MVVM architecture, Retrofit for API calls, Koin for dependency injection, and the Navigation Component for in-app routing. Published on the Google Play Store.
