A map-based tool for finding bicycle parking across Singapore. Data sourced from LTA DataMall (~2,542 locations), rendered with Mapbox GL JS and react-map-gl. Supports clustering for performance, address search via Mapbox Geocoding, shelter and rack type filters, and geolocation to find nearby spots.
