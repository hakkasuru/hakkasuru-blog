A mobile app for monitoring multiple GitLab instances. Tracks health status with color-coded indicators and countdown timers between checks. Supports adding, editing, and deleting instances with their API tokens, backed by a local SQLite database. Built with Flutter using go_router for navigation and Provider for state management.
