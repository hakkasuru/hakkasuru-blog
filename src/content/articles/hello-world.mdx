Welcome to my blog! This is the first post where I walk through the tech stack and decisions behind building this site.

## The Stack

- **Astro 5** for static site generation with islands architecture
- **Tailwind CSS v4** for utility-first styling
- **MDX** for rich content authoring
- **Vercel** for deployment

## Why Astro?

Astro's content-first approach was perfect for a blog. It ships zero JavaScript by default and lets me add interactivity only where needed — like the travel map.
